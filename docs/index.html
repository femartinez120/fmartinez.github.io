Introducción a la Programación Web

La programación web permite crear y desarrollar sitios y aplicaciones en Internet. Se compone de múltiples tecnologías, siendo las principales HTML, CSS y JavaScript.

Evolución Histórica

HTML

HTML (HyperText Markup Language) se introdujo en 1991 y ha evolucionado hasta HTML5, que soporta multimedia y estructuras semánticas. Desde sus inicios, HTML ha permitido la creación de páginas estáticas y con el tiempo ha incorporado etiquetas semánticas que mejoran la accesibilidad y estructura del contenido.

CSS

CSS (Cascading Style Sheets) apareció en 1996 para separar la presentación del contenido y mejorar la accesibilidad y diseño. CSS ha evolucionado significativamente con la introducción de CSS3, permitiendo transiciones, animaciones y un diseño más flexible a través de grid y flexbox.

JavaScript

JavaScript nació en 1995 y se ha convertido en un lenguaje clave para la interactividad en la web. Originalmente pensado para hacer validaciones simples en formularios, ha evolucionado hasta convertirse en un lenguaje de programación completo con soporte para manipulación del DOM, asincronismo y frameworks modernos como React y Vue.js.
